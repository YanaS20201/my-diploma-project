Вебзастосунок для покрокового переведення чисел між позиційними системами числення

Дипломний проєкт бакалавра за спеціальністю 121 "Інженерія програмного забезпечення".

Цей застосунок є інтерактивним математичним калькулятором, який дозволяє переводити довільні числа (включаючи дробові) між системами числення з основами від 2 до 36.
Головна особливість проєкту - повна візуалізація та деталізація кожного кроку розрахунку (ділення стовпчиком, множення дробів) з автоматичним контролем та оцінкою похибки обчислень.

Основний функціонал:
Універсальність: Переведення чисел із будь-якої системи числення (від 2 до 36) у будь-яку іншу в цьому ж діапазоні.
Робота з дробами: Точне якісне переведення раціональних дробів з урахуванням періодичності.
Покрокова візуалізація: Генерація детальних таблиць розрахунків для цілої та дробової частин числа (візуалізація алгоритмів "ділення стовпчиком" та "множення на основу").
Аналіз точності: Автоматичний розрахунок абсолютної та відносної похибок, якщо результат не може бути представлений точно в цільовій системі числення.

Технологічний стек:
Backend: Python 3.14+, FastAPI (для створення швидкого REST API), Uvicorn (ASGI-сервер).
Математичне ядро: Модуль `fractions` (для уникнення втрати точності за стандартом IEEE 754 під час проміжних обчислень).
Frontend: Чистий клієнтський JavaScript (ES6 Modules), HTML5, CSS3 (фреймворки навмисно не використовувалися для забезпечення максимальної швидкості завантаження та чистоти коду).
